# 扩展编写规范

## 组件

### 文件夹目录

组件下有 both, h5, pc 等文件夹,分别应用于多端, H5 端以及 PC 端等.



### 组件内部属性 name

每个组件都有一个内部属性 name, 根据以不同端进行命名.



- both  =>  'Lib-compName'
- h5     =>  'H5-compName'
- pc     =>  'pc-compName' 



### 组件内部函数命名

组件内部函数必须采用驼峰写法.

- 对于响应用户操的函数必须添加前缀 hand

- 对于自定义函数必须添加 func
- 对于 emit 事件名必须全用小写写法
